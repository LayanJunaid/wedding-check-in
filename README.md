# 💍 Wedding Check-in

A modern web application for managing wedding guest check-ins easily and efficiently.  
The system allows users to create weddings, add guests, send invitations, and track guest attendance in real time through a clean and simple dashboard.

🔗 **Live Demo:**  
https://wedding-check-in.vercel.app/login

🔗 **GitHub Repository:**  
https://github.com/LayanJunaid/wedding-check-in.git

## ✨ Features

- User authentication system  
- Create and manage multiple weddings  
- Add wedding date and wedding name  
- Add guests to each wedding  
- Track total guests, checked-in guests, and pending guests  
- Invite guests  
- Delete guests or weddings  
- Responsive and user-friendly interface  
- Deployed online using Vercel

## 📌 Project Pages

### 1. Login Page
Users can sign in using their email and password to access their wedding check-in dashboard.

### 2. My Weddings Page
After logging in, users can create a new wedding by entering the wedding name and date.  
All created weddings are displayed with options to manage or delete them.

### 3. Guests Management Page
Inside each wedding, users can add guest names, view guest status, invite guests, and delete guests when needed.

### 4. Dashboard Summary
The system displays a simple summary including:

- Total guests
- Checked-in guests
- Pending guests

## 🚀 How to Run the Project Locally

1. Clone the repository:

```bash
git clone https://github.com/LayanJunaid/wedding-check-in.git
```

2. Navigate to the project folder:

```bash
cd wedding-check-in
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
